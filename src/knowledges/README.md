# Snowflake 知识库

## 简介

本知识库用于存储和管理 Snowflake Index 项目相关的技术文档、学习资料和经验总结。

## 文档结构

- **技术文档**：项目相关的技术架构、设计方案和开发规范
- **学习资料**：AI学习相关的教程、文章和笔记
- **经验总结**：开发过程中的经验教训和最佳实践

## 如何使用

1. 在对应分类下创建新的文档
2. 文档命名使用清晰的描述性名称
3. 使用Markdown格式编写文档内容

## 示例文档

可以在此目录下创建更多专题文档，如：

- `ai学习笔记.md`
- `开发规范.md`
- `问题排查指南.md`
